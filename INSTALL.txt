Требования:
    - php 5.0 и выше
    - поддержка сессий в php
    - MySql
    - Веб-сервер

Для установки системы необходимо:
    - распаковать архив
    - настроить веб-сервер так, чтобы директория web была корневой
    - развернуть базу данных из дампа db_dump.sql
    - настроить файл config/config.php

Описание системы:
    - /?action=Login - окно авторизации
    - /?action=Register - регистрация
    - /?action=ModifyRegister - изменение профиля пользователя -TODO-
